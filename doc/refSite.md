- 다크 블루/블랙 베이스 + 화이트 텍스트 + 포인트 컬러
- 3단계 프로세스를 탭/스텝으로 시각화하는 패턴이 Service 페이지 참고

## 부록: Loomix AI vs 레퍼런스 사이트 비교 요약

| 구분 | Loomix AI | 주요 레퍼런스 |
|------|----------|-------------|

| **디자인 테마** | 다크 모드 전체 | Runway, Pika, Buck (다크) / 나머지는 라이트 |
| **Hero 패턴** | 풀스크린 비디오 | Runway, Pika |
| **포트폴리오** | 호버 비디오 프리뷰 + 다중 필터 | Buck (호버), InVideo (필터) |

| **Before/After** | 패러다임 전환 비교 | Colossyan |
| **다국어** | ko/en/ja 3개 + 런타임 전환 | HeyGen/Synthesia (100+언어는 영상 번역용, 사이트 UI는 영어) |
| **기술 스택** | Next.js 16 SSG + Tailwind 4 | 대부분 Webflow/WordPress/Custom React |
| **SEO** | Metadata + JSON-LD + hreflang | HeyGen, Synthesia (업계 최고 SEO) |
