## 반응형 브레이크포인트

| 브레이크포인트 | 너비        | 적용                                    |
| -------------- | ----------- | --------------------------------------- |
| 기본           | < 640px     | 단일 열, 모바일 레이아웃                |
| `sm`           | ≥ 640px     | 포트폴리오 2열, 버튼 가로 배치          |
| `lg`           | ≥ 1024px    | 포트폴리오 3열, 비즈니스 카드 2열       |
